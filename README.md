# DS8641N Tester
A simple tester for DS8641N chips.

![Chip Tester PCB](images/DS8641_tester.jpg "Chip Tester PCB")


## Usage ##
 1. Plug it into 5 volt power centre positive power.
 2. Install chip
 3. Press Buttons
